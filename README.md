## 🌐 Deployment (GitHub Pages)

1. Go to your GitHub repository
2. Click on **Settings**
3. Navigate to **Pages**
4. Under "Source", select:
   - Branch: `main`
   - Folder: `/root`
5. Click **Save**

Your site will be live at:
https://your-username.github.io/repo-name/
